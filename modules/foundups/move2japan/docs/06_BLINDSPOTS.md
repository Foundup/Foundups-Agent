# Move2Japan FoundUp — Blindspots & Outside the Box (012 Architecture Vision)

_Source: 012 voice-mode conversation, captured 2026-03-07_

## 1. Urgency Mode

Some stakeholders are panic-driven. Create an urgency classifier:

- Curiosity → Serious → Urgent → Distressed
- This changes agent tone and flow

## 2. Pathway Quiz

Fast diagnostic to determine likely route:

- Work visa path
- Student path
- Spouse/family path
- Remote income path
- Entrepreneur path
- Retirement/long-term stay exploration

## 3. Reality Gap Scoring

Some stakeholders want Japan but are not ready. System should estimate:

- Readiness, financial viability, timeline realism, language gap, documentation risk

## 4. Dream vs First Landing Split

**Critical distinction.** The stakeholder's dream life and first landing plan are not always the same.

- Dream: countryside home → First landing: Tokyo share house near job market

## 5. "Done for You" Document Lane (Premium)

- Translation help, draft completion, field-by-field prep, document inventory

## 6. Stakeholder Reactivation System

If a stakeholder stalls:

- Reminder email, dashboard nudge, "you're one step away" prompt, stage-specific encouragement

## 7. Success Archive

Anonymized stakeholder stories: how they moved, what path, timeline, cost, mistakes avoided

## 8. Japan Arrival Layer

Most relocation products stop at the move. Strong differentiator = **post-arrival help**:

- Ward office, bank, phone, transit card, insurance, apartment setup, local routines

## 9. Community Flywheel

Later: by city, by stage, by visa path, by profession

## 10. Trust Stack (Day One)

- Source dates on policy content
- "Not legal advice" boundaries
- Clear escalation points
- Secure account design
- Transparent partner disclosures

## Missing Layers Worth Designing Now

### Emotional State Layer

Some stakeholders are escaping fear, burnout, politics, instability, divorce, or reinvention.
Agent should detect: urgency, emotional overwhelm, fantasy vs readiness, practical seriousness.

### Trust Layer

Relocation is high-stakes. System needs: clear disclaimers, source-backed guidance, transparent human handoff, secure document handling, no fake certainty.

### Time Horizon Layer

Segments: curious, 12 months out, 6 months out, 90 days, urgent.

### Compliance Layer

Some guidance touches: immigration, employment, taxes, real estate, privacy, affiliate disclosure, document handling.

### Community Layer

Cohort by stage, city channels, newcomer circles, accountability groups, success stories.

### Re-entry Layer

Stakeholders disappear and return. System **must resume without restarting**.

### Partner Marketplace Layer (Later)

Brokers, visa specialists, translators, recruiters, movers, language coaches, local guides.
