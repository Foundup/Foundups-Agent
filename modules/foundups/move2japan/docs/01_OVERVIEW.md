# Move2Japan FoundUp — Overview (012 Architecture Vision)

_Source: 012 voice-mode conversation, captured 2026-03-07_

## Core Idea

Move2Japan is not an information dump. It is a **guided ascent**.

The stakeholder does not see the whole mountain. The system only reveals the next base camp.

- The live chat agent is the first gateway
- The web app is the persistent journey layer
- Premium agents and human partners unlock deeper execution

## Big Picture Outcome

Make it easy for a person to move to Japan by turning a complex relocation process into a staged roadmap with memory, guidance, automation, and optional human support.

## Core Principles

- **One stakeholder type, not customers** — participants are stakeholders in the system
- **Agent-first guidance** — AI orchestration drives the journey
- **Open-source modular architecture** — WSP-compliant, extensible
- **Base-camp progression, not overwhelm** — reveal only the next stage
- **Soft conversion, not hard selling** — build trust before monetization
- **Persistent memory across chat and web** — cross-session, cross-channel
- **Automate everything possible** — minimize manual labor
- **Human handoff only when high-trust or licensed support is needed**

## The Mountain Model

### Base Camp 0 — Intent Capture

The stakeholder types `!move2japan` in YouTube live chat.
System assesses urgency, captures passport status, and routes.

### Base Camp 1 — Identity and Readiness

Do they have a passport? If not, help them get one.

### Base Camp 2 — Migration Pathway

Why are they moving? Work, student, spouse, entrepreneur, remote worker, retiree, cultural relocation, escape/urgency.

### Base Camp 3 — Economic Viability

How will they live? Job, savings, remote income, business, sponsorship.

### Base Camp 4 — Location Fit

What is their dream life in Japan? Tokyo, Osaka, Fukuoka, countryside, beach town, ski town, family-friendly, low-cost, startup hub.

### Base Camp 5 — Housing Pathway

Rental, share house, employer housing, akiya, short-term landing strategy.

### Base Camp 6 — Paperwork and Execution

Visa docs, translated forms, timelines, checklists, reminders.

### Base Camp 7 — Landing and Settlement

Arrival prep, bank, phone, ward office, insurance, taxes, language ramp, local orientation.

## What the System Really Is

Move2Japan is a relocation operating system with:

- **Acquisition** via live chat
- **Memory** via stakeholder profile
- **Orchestration** via agents
- **Progression** via roadmap engine
- **Monetization** via premium automation and partner referrals
- **Retention** via updates, news, and personal progress

## Why It Works

Most people do not fail because information is unavailable. They fail because the process is:

- **Fragmented** — no single coherent path
- **Overwhelming** — too many decisions at once
- **Emotionally heavy** — fear, uncertainty, isolation

Move2Japan solves for: uncertainty, sequencing, motivation decay, paperwork friction, language barriers, and lack of personalized next-step guidance.

## Soft Conversion Model

The chat agent should not push hard. It should say:

- "Return here anytime"
- "Sign up if you want progress tracking"
- "Sign up if you want updates"
- "Sign up if you want a personal move agent"
- "Sign up if you want job/housing automation"

## FoundUp Positioning

> Move2Japan is a FoundUp: an open, agent-driven startup where participants are stakeholders in the system rather than passive customers.
