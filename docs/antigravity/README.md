# Antigravity Setup (FoundUps / WSP)

## Purpose
This folder provides the **Global Rules template** for Antigravity/Gemini in a WSP-governed FoundUps workspace.

## Install Global Rules

1. Copy `docs/antigravity/GEMINI.md` to your global rules path:
   - `~/.gemini/GEMINI.md`
2. In Antigravity, ensure **Global Rules** are enabled.

## Workspace Rules / Workflows (in this repo)

- Workspace rules live in `.agent/rules/`
- Workflows live in `.agent/workflows/`
