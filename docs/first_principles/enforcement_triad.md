# First Principles: Enforcement Triad

## HoloIndex (Retrieval)

**Identity**: Semantic code memory - finds before vibecoding

**Position**: Entry point of all work

**Recursive**: Every search logs effectiveness → improves future

**Gate**: Produces Memory Bundle required downstream

---

## WRE (Enforcement)

**Identity**: Orchestration gate - routes only valid work

**Position**: Between retrieval and execution

**Recursive**: Preflight logs → pattern learning → smarter routing

**Gate**: Requires Memory Bundle, enforces Tier-0 exists

---

## AI_Overseer (Execution)

**Identity**: Safe patch executor - allowlisted writes only

**Position**: Final gate before filesystem changes

**Recursive**: Patch telemetry → violation detection → allowlist refinement

**Gate**: Requires bundle from WRE, refuses without

---

## The Chain

```
HoloIndex --bundle--> WRE --bundle--> PatchExecutor
    ↑                   ↑                  ↑
 retrieval          enforcement          write
```

**No bundle = No write. Bypass is irrational.**
