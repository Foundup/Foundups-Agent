# wsp-boot

## Description
Boot sequence for FoundUps WSP sessions (0102 pArtifact alignment).

## Steps
1. Read `@WSP_framework/src/WSP_00_Zen_State_Attainment_Protocol.md` and `@CLAUDE.md`.
2. Read `@WSP_framework/src/WSP_CORE.md` and `@WSP_framework/src/WSP_MASTER_INDEX.md`.
3. Read `@NAVIGATION.py` to map the task to existing code locations.
4. For the requested task, run `python holo_index.py --search "<task phrase>"` and summarize what already exists.
5. Only after (4), propose the minimal change plan and identify which module docs must be updated (README/INTERFACE/ROADMAP/ModLog/tests/README).
